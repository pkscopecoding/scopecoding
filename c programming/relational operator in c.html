Relational Operators in c

Relational operators are used to compare values of two expressions. An expression that contains relational operators is called relational expression. If the relation is true then the value of relational expression is 1 and if the relation is false then the value of expression is 0.


the relational operators are :


operator Meaning
< less than
<= lessthan or equal to
== equalto
!= Not equal to
> greater than
>= greater than or equal to







Let us take an example:

Expression Relation Value of Expression
5 < 2 False 0
5 <= 2 False 0
5 == 2 False 0
5 != 2 True 1
5 > 2 True 1
5 >= 2 True 1
5 == 0 False 0
2 != 0 True 1
9 > 4 True 1






The relational operators are generally used in if...else and loop statements. Inside the if control statement evaluates an expression, and if this expression is true(non zero) then the next statement is executed, otherwise the next statement is skipped. The details of if statement are discussed in control statements.
