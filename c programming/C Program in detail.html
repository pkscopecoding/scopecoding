Rules of a C Program

Form of a C Program indicates how program has to be written. There are some rules to write all C programs .These are under:-

  • Each instructions in a C Program is written as a seperate statement.
  • The statements in a program must appear in the same order in which we wish them to be executed.
  • Blank spaces may be inserted between two words to improve the readability of the statement.
  • All statements should be in lower case letters.
  • C is called a free-form language because it has no specific rules for the position at which a statement is to be written in a given line.
  • Every C statement must end with a semicolon (;). Thus ; act as statement terminator.

What is main()

main() forms a crucial part of any C program. main() is a function. In C program, there can be multiple functions. Any C program should have atleast one function which should be main() . All statements that belong to main() are enclosed within a pair of braces { } as shown below.
  • int main()
    {
    statement1;
    statement2;
    statement3;
    }
Like function in a calculator, functions in C also return a value. main() function always returns an integer value, hence there is an int before main(). If for any reason the statements in main() fail to do their work, we can return a non- zero number from main(). This would indicate failure.
Some compiler like Turbo C/C++ allows us to return nothing from main(). In such cases we use keyword void . But its non- Standard way to write main() function.

Compilation and Execution

Once you have written the program, you need to type it and instruct the machine to execute it. Two other programs are needed to do this --- Editor and Compiler. Editor lets us type our program, whereas Compiler converts other program into machine language program. The conversion is necessary, since machine understand only machine language.

Apart from these two, there are other programs which you may need to improve your programming efficiency-- preprocessor, linker and Debugger. Working with each one of them individually is a tedious job. Hence, often all these are bundled together with a layer of GUI on top of them. This GUI makes using these programs eadier for your . This bundle is often called Integrated Deelopment Environmant(IDE). There are many IDEs available. Each is targeted towards different operating systems and microprocessors.
