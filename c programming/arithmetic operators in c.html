Arithmetic Operators

Arithmetic operators are used for numeric calculations. They are of two types-

1. Unary arithmetic operators

Unary operators require only one operand. For Example-
+x     -y
Here '-' Changes the sign of the operand y.

2. Binary arithmetic operators

Binary operators require Two operands(value on which an operation is to be done). There are five binary arithmetic operators-

Operator Purpose
+ addition
- subtraction
* multiplication
/ division
% Gives remainder in integer division

%(modulus operator) cannot be applied with floating point operands. There is no exponent operator in C, however there is a library function pow() to carry out exponentiation operation.

When both operands are integers, the resulting value is always an integer. Let us take two variables a and b. The value of a is 17 and b is 4 , the results of the following operations are-

Expression Result
a+b 21
a-b 13
a*b 68
a/b 4
a % b 1(remainder, integer division)

NOTE: After division operation the decimal part will be ignored and result is only integer part of quotient.
