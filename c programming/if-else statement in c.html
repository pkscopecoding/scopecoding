If- else statement

This is a bi-directional selection control statement which is used to take one of the two possible actions. The syntax of an if-else statement is:

if ( expression )
  statement1;
else
  statement2;

In other words we can say that :

if (expression) is true then,
  statement1;
if (expression) is false then,
  statement2;

The expression inside the parantheses is evaluated and if it is true (has a non-zero value), then statement1 is executed and if the expression is false (zero) then statement2 is executed. This expression is often called the if condition.



The else part in the if-else statement is optional , it means. an if-else statement can be written with or without the else clause. In case of missing else the syntax would be-

if (expression)
  statement1;

or

if (expression)
{
  statement;
 ................................
}

there can be a single statement or a block of statements after the if part. If the expression is true, then statement1 is executed and if it is false then the control transfers to the next statement which is immediately after the if control statement.


We can use boolean operators for combining conditions. For example, suppose we want to take some action if the value of variable is more than 10 and less than 25, then we can write an if statement as

if (var > 10 && var < 25)
    statement1

Now the statement1 will be executed only when both var > 10 and var < 25 are true.

Sometimes coding shortcuts are used while writing the conditions.

some shortcuts

1. if(exp!=0) is often written as if (exp)

The value of expression (exp!=0) will be true if exp is non-zero and false if exp is zero. The value of epression (exp) will be true if exp is non-zero(as non-zero is true in C) and false if exp is zero.

Hence, we can write if(exp) instead of if(exp!=0).

So whenever you see code if this form-

if(exp)
  statement1

read it as- if exp is non-zero execute statement1.


2... if(exp==0)   is often written as   if(!exp)

The value of expression (exp==0) will be true if exp is zero and false if exp is non-zero. The expression (!exp) will be true if exp is zero and false if exp is non-zero. Hence we can write if(!exp) instead of if(exp==0).

So whenever you see code if this form-

if(!exp)
  statement2
read it as -- if exp is zero execute statement2.
