Assignment Operator

Assignment operator is used to store value in a variable. The operand on the the left hand side should be a variable while the operand on the right hand side can be any variable (a,b,c,etc), constant(3,5,6,etc) or expression(a+b+3). The value of right hand operand is assigned to the left hand operand.
example:

  • x= 8
  • y = 5
  • s = x+y -2
  • y=x
  • x=y

  • // 8 is assigned to x
  • // 5 is assigned to y
  • // Value of expression x+y -2 is assigned to s
  • // value of x is assigned to y
  • // value of y is assigned to x

Assignment operator is giving (assigning ) value to any variable in expression. Lets take an example x=y=20. Here all three variables x, y, z will be assigned value 20, and the value of the whole expression will be 20.

example: We can also write :

x=x+5 can be written as x+=5 here += is a compound assignment operator.

Similarly we have other compound assignment operators:
Illustration:

  • x -= 5   is equivalent to   x=x-5
  • y*=5   is equivalent to   y=y*5
  • sum/=5   is equivalent to   sum=sum/5
  • k%=5+x   is equivalent to   k=k%(5+x)
  • a*=b+5   is equivalent to   a=a*(b+5)
