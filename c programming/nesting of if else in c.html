Nesting of if-else

We can have another if...else statement in the if part or the else part. This is called nesting of if....else statements. Here is an example of nesting where we have if....else inside both if and else parts.

if(expression)
{
  if (expression2)
    statementA1
  else
    statementA2
}
else
{
  if(expression3)
    statementB1
  else
    statementB2
}
