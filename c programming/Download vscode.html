How to download VS Code and C/C++ compiler

Here is the step by step process for downloading vs code for your machine. In this section we will also download c/c++ compiler so that it will be easier for you to run your code inside VS code easily.


Step1.

Go to Download vs code here and click on windows. The download will be started, when you click windows.

Step2.

open the vscodeusersetup exe file from downloads folder.

Step3.

You will see the same dialogue box as below, then click > i accept the agreement > Next

Step4.

If you want to change the destination location. then click browse and select any location where you want vs code to be installed. Otherwise click Next.

Step5.

click > Next.

Step6.

If you want to create a shortcut icon on desktop then check the box > create a destop icon. click Next.

Step7.

Click install to start the installation process

Step8.

Once the installation is completed. then click finish.

Congratulations. Vs code installed successfully.
