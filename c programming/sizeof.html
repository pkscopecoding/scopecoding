sizeof() Operator

sizeof is a unary operator that gives the size of its operand in terms of bytes. The operand can be a variable, constant or any datatype(int, float, char, etc.) , for example sizeof(int) gives the bytes occupied by the int datatype.


Generally sizeof operator is used to make portable programs ie. programs that can be run on different machines. For example if we write our program assumming int to be 2 bytes, then it won't run correctly on a machine on which int is of 4 bytes. So to make general code that can run on all machines we can use sizeof operator.
