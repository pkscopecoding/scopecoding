Setting Up VS Code for Python

I know you're pretty excited to get your hands on Python. But before starting, we must do some settings in order to make our life easier as a Python developer. First thing first we have to install some extensions in our Visual Studio Code. So, let's get into it.

Which Extensions to Install?

  • On your first time opening VS Code you will get this window. Don't mind if your appearance is different. I have set it to dark mode as I have installed VS Code earlier.
  • Go to Extensions on the left bottom and click it.
  • It will take you to the Visual Studio Extensions Marketplace.
  • On the search bar type Python and Install it.
  • Another Extension is Pylance. Type Pylance and then install it.
  • Now you're finally ready to get you hands on Python. Happy Coding.
