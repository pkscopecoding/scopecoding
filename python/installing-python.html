Installing Python

Here, we'll be learning to install Python on our local machine. The process of installation is more or less similar in different Operating System, but you can prefer the Python Docs if you get any error while installing Python. I'll be using Windows. So, let's get started.

Steps to Install Python

  • Go to the official Python website to download the latest Python version. Currently, it is Python 3.10.7 as of now.
  • Click on Python 3.10.7 and let the download complete.
  • Now, click on Install Now and make sure to tick both the boxes below.
  • Let the Installing Complete.
  • When the Installation is completed, open up Command Prompt and type "python --version". If this doesn't work type "py --version". Either of them work. In my case, I have already installed Python before, so while typing "python --version", it is showing me the earlier version.
  • Congratulation! You've successfully installed Python on your local machine.
