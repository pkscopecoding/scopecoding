Write Your First Program in Python

Finally, the moment has come where we'll be writing our first program on Visual Studio Code with Python. So, get yourself a cup of coffee and start our Python journey.

  • Create a Folder and name it as Python Lessons. I have created a folder in my Local Disk D.
  • Open Visual Studio Code and open the folder you've just created. It will look something like this.
  • Click on New File and name it as main.py. You will see a Python logo at first.
  • Let's write our first Python code.
  • Hit Ctrl + S to save the file and click on Run Code icon at top.
  • It will open the output tab below on which you can see your codes running.
  • Alternatively, you can also run the code by going into the Terminal to open up VS Code Terminal and run your python file by typing "python main.py" or in my case "py main.py". But make sure you are on the right folder or directory.
  • Congratulation! You've just wrote your first Python program. This is not the end here. There is so much more coming ahead.

In the next lesson, we'll be learning about print function and other things related to Python.
